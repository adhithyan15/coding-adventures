# lisp-compiler

Compiles Lisp ASTs to GenericVM bytecode with tail-call optimization

## Dependencies

- grammar-tools
- lexer
- parser
- virtual-machine
- bytecode-compiler
- garbage-collector
- lisp-lexer
- lisp-parser
- lisp-vm
- directed-graph
- state-machine

## Development

```bash
# Run tests
bash BUILD
```
