# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-23

### Added

- Initial package scaffolding generated by scaffold-generator
- Full MD5 implementation (RFC 1321) in `CodingAdventures.Md5`
  - `md5/1` — returns 16-byte raw binary digest
  - `md5_hex/1` — returns 32-character lowercase hex string
  - 64-round compression function with four stages (F/G/H/I auxiliary functions)
  - Correct little-endian word parsing via `<<w::little-32>>` Elixir bitstring matching
  - Correct little-endian output encoding via `<<a::little-32, b::little-32, c::little-32, d::little-32>>`
  - Correct little-endian 64-bit padding length field via `<<bit_len::little-64>>`
  - T-table of 64 sine-derived constants computed at compile time with `@t`
  - Shift amounts table `@s` as a module attribute
  - Davies-Meyer feed-forward after each block
- 38 unit tests covering RFC 1321 vectors, output format, little-endian correctness, block boundary edge cases, avalanche effect, and known hashes
- 100% test coverage
