# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-28

### Added

- Initial plugin implementation
- `.lattice` file transform hook with Lattice → CSS transpilation
- HMR (Hot Module Replacement) support for `.lattice` files
- Configurable options: minified output, indent string
- CSS injection via `<style>` tags with `data-lattice` attribute
