# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-29

### Added

- Core Caesar cipher operations: `encrypt`, `decrypt`, `rot13`
- Brute-force attack function returning all 25 possible decryptions
- Frequency analysis using chi-squared statistic against English letter frequencies
- English letter frequency table (`ENGLISH_FREQUENCIES`)
- `BruteForceResult` type for brute-force output
- Comprehensive test suite with 95%+ coverage
- Literate programming style with extensive JSDoc documentation
