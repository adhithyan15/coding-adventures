1 # CodingAdventures.AesModes — AES Modes of Operation
2 #
3 # AES operates on fixed 128-bit (16-byte) blocks. To encrypt messages longer
4 # than one block, you need a "mode of operation" that defines how multiple
5 # block cipher calls chain together. The choice of mode critically affects
6 # security:
7 #
8 # ECB — Electronic Codebook (INSECURE, educational only)
9 # CBC — Cipher Block Chaining (legacy, vulnerable to padding oracles)
10 # CTR — Counter mode (modern, stream cipher from block cipher)
11 # GCM — Galois/Counter Mode (authenticated encryption, gold standard)
12 #
13 # Why do modes matter?
14 # ────────────────────
15 # A raw block cipher is a fixed-width permutation: 16 bytes in, 16 bytes out.
16 # Real messages are longer. If you encrypt each block independently (ECB),
17 # identical plaintext blocks produce identical ciphertext blocks — the famous
18 # "ECB penguin" shows image structure leaking through encryption.
19 #
