# brainfuck-jvm-compiler

`brainfuck-jvm-compiler` is the end-to-end pipeline package for compiling
Brainfuck source into JVM `.class` bytes.

Pipeline:

1. `brainfuck` parses source into an AST
2. `brainfuck-ir-compiler` lowers Brainfuck into generic IR
3. `ir-optimizer` optionally optimizes the IR
4. `ir-to-jvm-class-file` lowers the IR into a JVM class artifact
5. `jvm-class-file` parses the generated bytes back as a structural check

## Quick Start

```python
from brainfuck_jvm_compiler import compile_source

result = compile_source("+" * 65 + ".")

assert result.class_name == "BrainfuckProgram"
assert result.class_bytes
assert result.parsed_class.this_class_name == "BrainfuckProgram"
```
