# Changelog

## [0.1.0] - 2026-04-16

### Added

- Initial implementation of `coding-adventures-beam-opcode-metadata`.
- `BeamOpcode` and `BeamProfile` dataclasses for version-aware external BEAM opcode metadata.
- Built-in OTP release profiles for `otp24`, `otp28`, and `otp29`.
- Full external opcode catalog through opcode `191`.
- Helpers for profile lookup and profile enumeration.
- Typed package exports and unit tests for profile and opcode lookup behavior.
