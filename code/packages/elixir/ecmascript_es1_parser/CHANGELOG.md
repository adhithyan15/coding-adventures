# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-05

### Added

- Initial implementation of ECMAScript 1 (1997) parser
- Grammar-driven parsing using es1.grammar
- Full ES1 statement and expression support
- Convenience functions: `parse/1`, `load_grammar/0`, `grammar_path/0`
