# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-18

### Added

- Initial package scaffolding generated by scaffold-generator
- Repository-owned storage types for named records, pages, stats, and leases
- `StorageBackend` trait with point reads, conditional writes, prefix listing,
  and advisory leases
- `InMemoryStorageBackend` as a reusable pure Rust backend for tests, examples,
  and higher-level store development
- Validation helpers for namespaces, keys, revisions, and content types
- Backend conformance helpers for shared storage backend test suites
