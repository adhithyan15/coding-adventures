# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-04

### Added

- Initial package scaffolding generated by scaffold-generator
- `parseMosaic(source)` — parses `.mosaic` source into an ASTNode tree (root rule: `"file"`)
- `PARSER_GRAMMAR` — pre-compiled `ParserGrammar` constant with all 18 Mosaic grammar rules
- `_grammar.ts` — embedded parser grammar: file, import_decl, component_decl, slot_decl, slot_type, list_type, default_value, node_tree, node_element, node_content, property_assignment, property_value, slot_ref, enum_value, child_node, slot_reference, when_block, each_block
- Full test suite: 42 tests covering all grammar rules, slot types, property values, when/each blocks, imports, and error cases
