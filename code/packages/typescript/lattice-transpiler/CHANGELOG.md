# Changelog

All notable changes to this package will be documented in this file.

## [0.1.1] - 2026-03-25

### Fixed

- **Embedded grammar strings updated** — `browser.ts` embeds verbatim copies of the
  Lattice grammar files for browser-compatible transpilation. Three grammar bugs were
  fixed in the shared `.grammar`/`.tokens` files but the copies in `browser.ts` were
  never updated, so the playground continued to fail after the logic-level fixes.
  Updated `LATTICE_TOKENS_GRAMMAR` and `LATTICE_PARSER_GRAMMAR` to match:
  - `LESS = "<"` token added so `@if $val < $min` no longer throws a lexer error.
  - `comparison_op` now includes `LESS` so `<` is a valid comparison operator.
  - `lattice_multiplicative` now includes `SLASH` so `$px / 16 * 1rem` parses.
  - `mixin_param` now uses `mixin_value_list` (COMMA-excluded) so multi-param
    mixin defaults like `@mixin card($bg: #fff, $shadow: true)` parse correctly.
