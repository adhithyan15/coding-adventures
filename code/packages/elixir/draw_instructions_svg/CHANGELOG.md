# Changelog

All notable changes to this package will be documented in this file.

## [0.2.0] - 2026-03-31

### Added

- Line rendering (`<line>` SVG element) for grid lines and separators
- Clip rendering (`<clipPath>` + `<g clip-path="...">`) for constraining children
- Stroke support on rectangles (stroke and stroke-width attributes)
- Font weight support on text (font-weight attribute, only emitted when bold)
- `@behaviour CodingAdventures.DrawInstructions` declaration
- Deterministic clip IDs using process dictionary counter (resets per render)
- Comprehensive literate-programming documentation

### Changed

- Empty metadata maps no longer emit any `data-*` attributes
- Clip ID counter resets at the start of each `render/1` call for deterministic output
