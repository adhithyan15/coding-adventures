# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-22

### Added
- `Message` class with immutable design, binary wire format, factory methods
- `Channel` class with append-only log, binary persistence, crash recovery
- `Actor` class with mailbox, behavior functions, state management
- `ActorSystem` runtime with lifecycle management, message delivery, dead letters
- Comprehensive test suite with 58 test cases targeting 80%+ coverage
