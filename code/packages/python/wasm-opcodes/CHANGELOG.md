## [0.1.1] - 2026-03-23

### Added

- Full implementation of the `wasm-opcodes` package
- `OpcodeInfo` frozen dataclass with fields: `name`, `opcode`, `category`,
  `immediates`, `stack_pop`, `stack_push`
- `OPCODES: dict[int, OpcodeInfo]` — lookup by byte value
- `OPCODES_BY_NAME: dict[str, OpcodeInfo]` — lookup by mnemonic
- `get_opcode(byte)` — returns `OpcodeInfo | None` for a given byte value
- `get_opcode_by_name(name)` — returns `OpcodeInfo | None` for a given mnemonic
- Complete table of all 172 WASM 1.0 instructions across 9 categories:
  control, parametric, variable, memory (loads, stores, management),
  numeric_i32, numeric_i64, numeric_f32, numeric_f64, conversion
- 74 unit tests with 100% statement coverage
- Literate programming: inline explanations of structured control flow,
  memarg encoding, stack effects, and conversion naming conventions
- `BUILD_windows` file for Windows CI
