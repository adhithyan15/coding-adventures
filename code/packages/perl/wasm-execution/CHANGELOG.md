# Changelog

All notable changes to this package will be documented in this file.

## [0.2.0] - 2026-04-05

### Added

- Complete WASM 1.0 instruction handler set:
  - i32: clz, ctz, popcnt, div_u, rem_s, rem_u, shr_u, rotl, rotr
  - i32 comparisons: lt_u, gt_s, gt_u, le_s, le_u, ge_s, ge_u
  - i64 comparisons: eqz, eq, ne, lt_s, gt_s, le_s, ge_s
  - i64 arithmetic: div_s, and, or, xor, shl
  - Memory stores: i64.store, f32.store, f64.store
  - f32/f64: abs, neg, div
  - Conversions: i32.trunc_f64_s, i64.extend_i32_u, f32.convert_i32_s/u,
    f64.convert_i32_s/u, f64.convert_i64_s, i32.reinterpret_f32,
    f32.reinterpret_i32
- Comprehensive test suite covering typed values, linear memory, tables,
  bytecode decoder, constant expression evaluator, and engine execution
  (29 test cases)

### Fixed

- Fixed `store_i32_8` pack format to use unsigned `'C'` instead of signed
  `'c'` to avoid "Character wrapped" warnings on Perl 64-bit
