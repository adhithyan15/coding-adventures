# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-05

### Added

- Initial implementation of ECMAScript 5 (2009) lexer
- Grammar-driven tokenization using es5.tokens
- Support for debugger keyword and all ES3 features
- Convenience functions: `tokenize/1`, `load_grammar/0`, `grammar_path/0`
