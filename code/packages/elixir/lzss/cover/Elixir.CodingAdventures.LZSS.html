5 LZSS lossless compression algorithm (1982) — CMP02.
6
7 LZSS (Lempel-Ziv-Storer-Szymanski) refines LZ77 by eliminating the mandatory
8 `next_char` byte appended after every token. Instead, a flag-bit scheme
9 distinguishes literals from back-references:
10
11 - Literal → 1 byte (flag bit = 0)
12 - Match → 3 bytes (flag bit = 1: offset uint16 BE + length uint8)
13
14 Tokens are grouped in blocks of 8. Each block starts with a flag byte
15 (bit 0 = first token, bit 7 = eighth token).
16
17 ## Wire Format (CMP02)
18
19 Bytes 0-3: original_length (big-endian uint32)
20 Bytes 4-7: block_count (big-endian uint32)
21 Bytes 8+: blocks
22 Each block: [1-byte flag] [1 or 3 bytes per symbol]
23
24 ## Series
25
26 CMP00 (LZ77, 1977) — Sliding-window backreferences.
27 CMP01 (LZ78, 1978) — Explicit dictionary (trie).
28 CMP02 (LZSS, 1982) — LZ77 + flag bits. ← this module
29 CMP03 (LZW, 1984) — LZ78 + pre-initialised alphabet; GIF.
30 CMP04 (Huffman, 1952) — Entropy coding.
31 CMP05 (DEFLATE, 1996) — LZ77 + Huffman; ZIP/gzip/PNG/zlib.
