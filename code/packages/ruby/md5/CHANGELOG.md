# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-23

### Added

- Initial package scaffolding generated by scaffold-generator
- Full MD5 algorithm implementation (RFC 1321) in `lib/coding_adventures_md5.rb`:
  - `CodingAdventures::Md5.md5(data)` — one-shot 16-byte binary digest
  - `CodingAdventures::Md5.md5_hex(data)` — one-shot 32-char hex digest
  - `CodingAdventures::Md5::Digest` — streaming hasher class with `update`, `digest`, `hexdigest`, `copy`, and `<<` alias
- 54 tests covering:
  - All 7 RFC 1321 official test vectors
  - Output format (binary encoding, length, hex format)
  - Little-endian byte-order verification
  - Block boundary cases (55, 56, 63, 64, 128 bytes)
  - Edge cases (null bytes, binary data, UTF-8 bytes)
  - Streaming API: chained updates, block-crossing updates, one-byte-at-a-time
  - Digest non-destructiveness (digest() does not alter state)
  - Copy/branch semantics
  - Large inputs (1000 bytes with chunked streaming)
  - Known-value spot checks (Hello World, quick brown fox)
- Literate programming style: T-table derivation, little-endian vs big-endian explanation, stage auxiliary function truth tables, Davies-Meyer feed-forward rationale — all inline in source
- `CodingAdventures::Md5::Core` internal module encapsulates `compress`, `pad`, and `rotl` so that `Digest` can access them while keeping them out of the public API surface
- Verified all results against Ruby stdlib `Digest::MD5`
