# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-23

### Added

- Initial package scaffolding generated by scaffold-generator
- Implemented `DecodeUnsigned(data []byte, offset int) (uint64, int, error)` with
  `*LEB128Error` on unterminated/malformed input
- Implemented `DecodeSigned(data []byte, offset int) (int64, int, error)` with
  two's complement sign extension
- Implemented `EncodeUnsigned(value uint64) []byte`
- Implemented `EncodeSigned(value int64) []byte` for signed two's complement values
- `LEB128Error` struct with `Message string` and `Offset int`, implementing the
  `error` interface
- 41 unit tests covering all 11 required cases:
  zero, one-byte unsigned, one-byte signed negative, multi-byte (624485),
  max u32, max i32, min i32, round-trips, unterminated error, non-zero offset,
  and signed negative values
- Test coverage: 96.0% (exceeds 80% threshold)
- Literate programming style — full algorithm explanations, bit-layout diagrams,
  worked examples, and inline derivations inside every function and its doc comment
- `BUILD` file for the build tool
