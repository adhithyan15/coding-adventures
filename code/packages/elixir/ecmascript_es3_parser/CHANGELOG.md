# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-05

### Added

- Initial implementation of ECMAScript 3 (1999) parser
- Grammar-driven parsing using es3.grammar
- try/catch/finally/throw, strict equality, instanceof support
- Convenience functions: `parse/1`, `load_grammar/0`, `grammar_path/0`
