# Changelog

All notable changes to this package will be documented in this file.

## [0.2.0] - 2026-03-23

### Added

- `lattice_transpiler.go` — Full implementation of the end-to-end pipeline
  wrapper with three public entry points:
  - `Transpile(source)` — pretty-printed CSS with 2-space indentation
  - `TranspileMinified(source)` — compact CSS with no extra whitespace
  - `TranspileWithOptions(source, opts)` — full control via `Options` struct
  - `Options` struct with `Minify bool` and `Indent string` fields
  - `resolveIndent` helper: empty `Indent` defaults to `"  "` (2 spaces);
    `Indent` is ignored when `Minify` is true

- `lattice_transpiler_test.go` — Comprehensive test suite (36 tests) covering:
  - `Transpile`: plain CSS passthrough, variable substitution, pretty-print
    defaults, mixin expansion, `@if`/`@else`, function calls
  - `TranspileMinified`: basic minification, variable resolution in minified mode
  - `TranspileWithOptions`: zero-value defaults, minify flag, custom indent,
    4-space indent, minify overrides indent
  - `resolveIndent`: all four combinations of (Minify, Indent) inputs
  - Error propagation: `UndefinedVariableError`, `UndefinedMixinError`,
    `WrongArityError`, lexer error resilience
  - Integration tests (spec scenarios): function in value, `@each` loop,
    `@if` inside `@each`, scope shadowing, CSS passthrough, `@media` rule,
    empty source, mixin with defaults, `@for` loop, circular mixin detection
  - Coverage: 100%
