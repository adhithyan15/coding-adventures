# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-23

### Added

- Initial package scaffolding generated by scaffold-generator
- Implemented `decode_unsigned(data, offset=0) → (value, bytes_consumed)` with
  `LEB128Error` on unterminated/malformed input
- Implemented `decode_signed(data, offset=0) → (value, bytes_consumed)` with
  two's complement sign extension
- Implemented `encode_unsigned(value) → bytes` (raises `ValueError` for
  negative input)
- Implemented `encode_signed(value) → bytes` for signed two's complement values
- `LEB128Error` exception class with `message` and `offset` attributes
- 95 unit tests covering all 11 required cases:
  zero, one-byte unsigned, one-byte signed negative, multi-byte (624485),
  max u32, max i32, min i32, round-trips, unterminated error, non-zero offset,
  and signed negative values
- Test coverage: 96.88% (exceeds 95% target)
- Literate programming style — full algorithm explanations, bit-layout diagrams,
  worked examples, and truth tables inline with every function
- `BUILD` (Linux/macOS) and `BUILD_windows` files for the build tool
