# Power Supply

Electrical power source interfaces scaling pure physical waves into simulated energy outputs.

## Overview

The `power-supply` package establishes constraints and parameters outlining exactly what type of electricity models an underlying circuit structure receives before moving into solver pipelines.

It provides distinct and explicit power delivery wrappers including `IdealDCSupply` and `IdealSinusoidalSource`.

## Dependencies
- Bridges the abstract properties of `analog-waveform` directly into an energy interface context.
