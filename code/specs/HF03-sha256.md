# HF03 — SHA-256

## Overview

SHA-256 is a cryptographic hash function from the SHA-2 family, designed
by the NSA and published by NIST in 2001 (FIPS 180-2). It produces a
256-bit (32-byte) digest and is the workhorse of modern cryptography —
used in TLS, Bitcoin, git, code signing, and password hashing.

Unlike MD5 (broken 2004) and SHA-1 (broken 2017), SHA-256 remains secure
with no known practical attacks. The birthday bound is 2^128, making
collision search computationally infeasible.

SHA-256 follows the Merkle-Damgård construction like MD5 and SHA-1, but
with a wider state (8 × 32-bit words), more rounds (64), and a more
complex message schedule.

## Algorithm

### Constants
- 8 initial hash values H0..H7 (first 32 bits of fractional parts of
  square roots of first 8 primes: 2, 3, 5, 7, 11, 13, 17, 19)
- 64 round constants K0..K63 (first 32 bits of fractional parts of
  cube roots of first 64 primes)

### Padding (Merkle-Damgård)
1. Append bit `1` (byte 0x80)
2. Append zeros until length ≡ 448 mod 512 (56 mod 64 bytes)
3. Append original message length as 64-bit **big-endian** integer

### Compression (per 512-bit block)
1. Parse block into 16 × 32-bit words W0..W15 (big-endian)
2. Extend to 64 words: W[t] = σ1(W[t-2]) + W[t-7] + σ0(W[t-15]) + W[t-16]
3. Initialize working variables a..h from current hash state
4. 64 rounds: T1 = h + Σ1(e) + Ch(e,f,g) + K[t] + W[t]
              T2 = Σ0(a) + Maj(a,b,c)
              Rotate and update
5. Add working variables to hash state

### Auxiliary Functions
- Ch(x,y,z) = (x AND y) XOR (NOT x AND z)
- Maj(x,y,z) = (x AND y) XOR (x AND z) XOR (y AND z)
- Σ0(x) = ROTR(2,x) XOR ROTR(13,x) XOR ROTR(22,x)
- Σ1(x) = ROTR(6,x) XOR ROTR(11,x) XOR ROTR(25,x)
- σ0(x) = ROTR(7,x) XOR ROTR(18,x) XOR SHR(3,x)
- σ1(x) = ROTR(17,x) XOR ROTR(19,x) XOR SHR(10,x)

## Interface Contract

| Function | Signature | Description |
|----------|-----------|-------------|
| `sha256` | `(data: bytes) -> bytes` | Returns 32-byte digest. |
| `sha256_hex` | `(data: bytes) -> string` | Returns 64-char lowercase hex. |
| `SHA256Hasher` | class/struct | Streaming hasher. |
| `.update` | `(data: bytes) -> self` | Feed bytes (chainable). |
| `.digest` | `() -> bytes` | Get 32-byte result (non-destructive). |
| `.hex_digest` | `() -> string` | Get 64-char hex result. |
| `.copy` | `() -> SHA256Hasher` | Deep copy for branching. |
