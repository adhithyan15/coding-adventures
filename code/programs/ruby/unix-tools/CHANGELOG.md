# Changelog

All notable changes to the `unix-tools` package (Ruby) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-21

### Added

- Restructured from standalone `pwd` program into `unix-tools` collection
- `pwd` tool: reimplementation of POSIX `pwd` powered by CLI Builder
  - `-L` / `--logical` flag: print logical working directory (default)
  - `-P` / `--physical` flag: print physical working directory (resolve symlinks)
  - `--help` and `--version` auto-generated by CLI Builder from `pwd.json`
  - POSIX-compliant fallback: `-L` falls back to physical path if `$PWD` is unset or stale
  - Full test suite exercising CLI Builder integration and business logic
