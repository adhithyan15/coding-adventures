3 # AES (Advanced Encryption Standard) is the most widely deployed symmetric
4 # encryption algorithm in the world. Published by NIST in 2001 as FIPS 197,
5 # it replaced DES and is used in TLS/HTTPS, WPA2/WPA3 WiFi, disk encryption
6 # (BitLocker, LUKS, FileVault), VPNs, and virtually every secure protocol.
7 #
8 # Designed by Joan Daemen and Vincent Rijmen (algorithm name: Rijndael), AES
9 # is a Substitution-Permutation Network (SPN) — structurally different from
10 # DES's Feistel network. In an SPN, ALL bytes of the state are transformed on
11 # every round, not just half.
12 #
13 # Architecture
14 # ─────────────
15 # plaintext (16 bytes)
16 # │
17 # AddRoundKey(state, round_key[0]) ← XOR first key material
18 # │
19 # ┌── Nr-1 full rounds ──────────────────────────────────────────────────┐
20 # │ SubBytes — non-linear S-box (GF inverse + affine transform) │
21 # │ ShiftRows — cyclic left shifts of each row (column diffusion) │
22 # │ MixColumns — GF(2^8) matrix multiply (row diffusion) │
23 # │ AddRoundKey — XOR with round key │
24 # └──────────────────────────────────────────────────────────────────────┘
25 # │
26 # SubBytes + ShiftRows + AddRoundKey ← final round (no MixColumns)
27 # │
28 # ciphertext (16 bytes)
29 #
30 # GF(2^8) polynomial: 0x11B = x^8 + x^4 + x^3 + x + 1
31 # (Different from Reed-Solomon's 0x11D; we implement arithmetic inline.)
32 #
33 # State Layout: column-major 4×4 bytes
34 # state[row][col] = block[row + 4*col]
35 #
36 # Key Sizes and Round Counts
37 # ──────────────────────────
38 # Key Nk Nr Round keys
