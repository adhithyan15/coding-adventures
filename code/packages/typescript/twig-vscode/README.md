# Twig for VS Code

Twig language support for Visual Studio Code.

## Features

- Language intelligence (powered by `twig-lsp-server`)
- Debugging support (powered by `twig-dap`)

## Prerequisites

You must install the Twig backend binaries separately and have them
available on your PATH (or override their paths in VS Code settings):

- `twig-lsp-server` — language server
- `twig-dap` — debug adapter


If your binaries are installed somewhere off PATH, override:

```jsonc
{
  "twig.serverPath": "/absolute/path/to/twig-lsp-server",
  "twig.adapterPath": "/absolute/path/to/twig-dap"
}
```

## Development

```bash
npm install
npm run build      # tsc compiles src/ → out/
```

To install the extension into VS Code locally, package it with
`vsce package` and use `code --install-extension`.
