# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-18

### Added

- Pure C# Huffman tree package with deterministic greedy construction over the existing C# heap package
- Code-table, canonical-code, decoding, inspection, and invariant-check helpers for the DT27 teaching surface
- xUnit coverage for validation, single-symbol trees, canonical ordering, decode behavior, and structural invariants
- BUILD scripts that isolate `.NET` artifacts and first-run state for Linux and Windows CI
