# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-18

### Added

- `LogicBytecodeOp` and `LogicBytecodeInstruction` for the first logic loader bytecode
- `compile_program()` for lowering LP07 instruction streams into bytecode plus pools
- `decode_program()` for bytecode-to-instruction round-trips
- disassembly helpers for readable bytecode inspection
