# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-12

### Added

- Initial package scaffolding generated by scaffold-generator
- `CodingAdventures::UrlParser::Url.parse` — single-pass left-to-right URL parser
- `CodingAdventures::UrlParser::Url#resolve` — RFC 1808 relative URL resolution
- `CodingAdventures::UrlParser::Url#effective_port` — scheme-default port lookup
- `CodingAdventures::UrlParser::Url#authority` — authority string reconstruction
- `CodingAdventures::UrlParser::Url#to_url_string` / `#to_s` — URL serialization
- `CodingAdventures::UrlParser.percent_encode` — RFC 1738 percent-encoding
- `CodingAdventures::UrlParser.percent_decode` — percent-decoding with UTF-8 support
- Error hierarchy: MissingScheme, InvalidScheme, InvalidPort, InvalidPercentEncoding, EmptyHost, RelativeWithoutBase
- Support for scheme:path URLs (mailto:, data:)
- IPv6 address parsing with bracket notation
- Dot segment removal (`.` and `..`) per RFC 3986 section 5.2.4
- 60 minitest tests with 109 assertions (ported from Rust implementation)
