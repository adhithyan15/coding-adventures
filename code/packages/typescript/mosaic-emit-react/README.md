# mosaic-emit-react

React backend for the Mosaic compiler pipeline. Takes a `MosaicIR` component
description and emits a `.tsx` file containing a typed React functional
component.

## Where it fits in the pipeline

```
.mosaic source
    └─► mosaic-lexer  (tokens)
    └─► mosaic-parser  (ASTNode tree)
    └─► mosaic-analyzer  (MosaicIR — typed IR)
    └─► mosaic-vm  (drives traversal, calls renderer)
    └─► mosaic-emit-react  ← YOU ARE HERE
              └─► ProfileCard.tsx
```

## Usage

```typescript
import { MosaicReactRenderer } from "@coding-adventures/mosaic-emit-react";
import { MosaicVM } from "@coding-adventures/mosaic-vm";
import { analyzeMosaic } from "@coding-adventures/mosaic-analyzer";

const ir = analyzeMosaic(source);          // parse + analyze .mosaic source
const vm = new MosaicVM(ir);
const result = vm.run(new MosaicReactRenderer());

// result.files[0].filename  → "ProfileCard.tsx"
// result.files[0].content   → "// AUTO-GENERATED ..."
```

## What gets generated

Given a Mosaic component like:

```mosaic
component ProfileCard {
  slot name: text
  slot age: number
  slot avatar: image

  Box {
    padding: 16dp;
    Image {
      src: @avatar;
    }
    Text {
      content: @name;
    }
  }
}
```

The renderer produces a `ProfileCard.tsx` file containing:

```tsx
// AUTO-GENERATED by mosaic-emit-react — do not edit by hand
import React from "react";

interface ProfileCardProps {
  name: string;
  age: number;
  avatar: string;
}

export function ProfileCard({ name, age, avatar }: ProfileCardProps): React.ReactElement {
  return (
    <div style={{ display: "flex", padding: "16px" }}>
      <img src={avatar} alt="" />
      <span>{name}</span>
    </div>
  );
}
```

## Output conventions

| Mosaic concept | React output |
|---|---|
| `Box` | `<div>` |
| `Text` | `<span>` |
| `Image` | `<img>` |
| `Button` | `<button>` |
| `slot foo: text` | `foo: string` in Props interface |
| `slot bar: number` | `bar: number` in Props interface |
| `slot baz: bool` | `baz: boolean` in Props interface |
| `slot qux: node` | `qux: React.ReactNode` in Props interface |
| `slot xs: list<text>` | `xs: string[]` in Props interface |
| `@slotName` in property | `{slotName}` JSX expression |
| `when @flag { ... }` | `{flag && (...)}` |
| `each @items as item { ... }` | `{items.map((item, _index) => (...))}` |
| `a11y-role: button` | `role="button"` |
| `a11y-role: heading` | renders as `<h2>` |
| `a11y-hidden: true` | `aria-hidden="true"` |
| `color: #rrggbbaa` | `rgba(r, g, b, alpha)` in `style` |
| `padding: 16dp` | `padding: "16px"` in `style` |

Styles are passed as an inline `style` object. Class names from `class` and
`text-align` properties are applied via `className`.

- `@coding-adventures/parser` — base parser
