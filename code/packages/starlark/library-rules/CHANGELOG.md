# Changelog

All notable changes to the library-rules package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-22

### Added
- `python_library.star` — py_library() rule for Python library packages
- `go_library.star` — go_library() rule for Go library packages
- `ruby_library.star` — ruby_library() rule for Ruby library packages
- `typescript_library.star` — ts_library() rule for TypeScript library packages
- `rust_library.star` — rust_library() rule for Rust library packages
- `elixir_library.star` — elixir_library() rule for Elixir library packages
- Knuth-style literate programming documentation in all rule files
- BUILD file, README.md, and CHANGELOG.md
