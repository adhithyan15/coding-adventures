# ml-learning-visualizer

Interactive machine learning training visualizer

## What It Shows

This renderer app visualizes a tiny linear regression model learning the
Celsius-to-Fahrenheit relationship:

```text
F = weight * C + bias
```

You can tune the learning rate, initial weight, initial bias, and loss function,
then step through gradient descent while the fitted line, prediction points,
loss, gradients, and error distances update.
