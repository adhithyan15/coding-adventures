# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-22

### Added
- Message class with immutable properties, binary wire format, convenience constructors
- Channel class with append-only log, binary persistence, crash recovery
- Actor class with mailbox, behavior function, state management
- ActorSystem runtime with lifecycle management, message delivery, dead letters
- 58 comprehensive test cases targeting 95%+ coverage
