## [0.1.0] - 2026-04-25

### Added

- Initial package scaffolding generated by scaffold-generator

### Implemented

- `LCG` class: Linear Congruential Generator (Knuth 1948) with full 2^64 period.
  Constants: multiplier = 6364136223846793005, increment = 1442695040888963407.
  Output is upper 32 bits of state.
- `Xorshift64` class: Marsaglia (2003) XOR-shift generator with period 2^64 − 1.
  Three shifts (13, 7, 17). Seed 0 replaced with 1. Output is lower 32 bits.
- `PCG32` class: O'Neill (2014) Permuted Congruential Generator. LCG recurrence
  plus XSH RR output permutation. Two-step initseq warm-up.
- All three classes implement: `next_u32`, `next_u64`, `next_float`, `next_int_in_range`.
- `next_int_in_range` uses rejection sampling to eliminate modulo bias.
- Reference values match the Go implementation exactly for seed=1:
  - LCG:        [1817669548, 2187888307, 2784682393]
  - Xorshift64: [1082269761, 201397313,  1854285353]
  - PCG32:      [1412771199, 1791099446, 124312908]
- 41 busted tests covering construction, reference values, determinism,
  output ranges, and full distribution of `next_int_in_range`.
