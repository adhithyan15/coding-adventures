# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-18

### Added

- `LogicBytecodeVM` and runtime state for direct execution of LP09 bytecode
- bytecode helpers for single-query, all-query, and compile-and-run execution
- bytecode validation for malformed opcodes, operands, pool indexes, and halt placement
- tracing support for step-by-step loader-bytecode execution
