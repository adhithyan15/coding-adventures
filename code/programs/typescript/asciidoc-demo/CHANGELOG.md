# Changelog

## 0.1.0 — Initial release

- Split-pane live preview: AsciiDoc editor (left) + HTML preview (right).
- Real-time rendering on every keystroke via @coding-adventures/asciidoc.
- Default example content showcasing all major AsciiDoc features.
- Footer stats: character count, word count, render time (ms).
- Dark theme with orange accent color to distinguish from the CommonMark demo.
- Error boundary: parse errors display inline without crashing the app.
- Tab key inserts two spaces instead of moving focus.
- Responsive: stacks vertically on narrow screens.
