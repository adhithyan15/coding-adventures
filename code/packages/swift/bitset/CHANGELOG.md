# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-06

### Added
- Initial implementation of `Bitset` conforming to the foundational bitset specification.
- Dynamic array-list style sizing mechanism with 64-bit word chunks.
- Basic bit operation logic for logic and arithmetic layers downstream.
