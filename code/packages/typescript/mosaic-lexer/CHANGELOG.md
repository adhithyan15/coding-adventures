# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-04

### Added

- Initial package scaffolding generated by scaffold-generator
- `tokenizeMosaic(source)` — tokenizes `.mosaic` source using embedded token grammar
- `TOKEN_GRAMMAR` — pre-compiled `TokenGrammar` constant (no disk I/O)
- `_grammar.ts` — embedded token definitions for all 16 Mosaic keywords, IDENT with hyphens, DIMENSION, NUMBER, COLOR_HEX, STRING, and all delimiters
- Full test suite: 14 test groups covering all token types, skip patterns, position tracking, and ordering invariants
