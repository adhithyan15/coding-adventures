# Energy Efficiency Dataset

This example includes `energy-efficiency.csv`, converted from the UCI Energy
Efficiency dataset file `ENB2012_data.xlsx`.

Source:
https://archive.ics.uci.edu/ml/datasets/Energy%2Befficiency

Citation:
Tsanas, A. & Xifara, A. (2012). Energy Efficiency [Dataset]. UCI Machine
Learning Repository. https://doi.org/10.24432/C51307.

License:
Creative Commons Attribution 4.0 International (CC BY 4.0).

Dataset shape used here:

- 768 samples
- 8 input features: `X1..X8`
- 2 output targets: `Y1` heating load and `Y2` cooling load
