# Local Dataset Sources

This app should not load training data from remote services at runtime. Small teaching datasets can be checked in only when their source and license are clear.

## Palmer Penguins Sample

- Source: https://github.com/allisonhorst/palmerpenguins
- Hugging Face mirror: https://huggingface.co/datasets/SIH/palmer-penguins
- License: CC0 1.0 Universal
- Citation: Horst AM, Hill AP, Gorman KB (2020). palmerpenguins: Palmer Archipelago (Antarctica) penguin data.
- Included file: `palmer-penguins.csv`
- Note: This is a compact teaching sample containing the first complete rows from the simplified `penguins.csv` table.
