# Changelog

## 0.1.0 — 2026-04-05

- Initial release
- Tokenizes ECMAScript 1 (1997) source code using the grammar-driven lexer
- Loads `es1.tokens` grammar file for the first standardized JavaScript edition
- Supports all ES1 keywords, operators, literals, and delimiters
- Thin wrapper around `@coding-adventures/lexer`
