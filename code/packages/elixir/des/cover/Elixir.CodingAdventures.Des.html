1 # CodingAdventures.Des — DES and 3DES block cipher (FIPS 46-3 / SP 800-67)
2 #
3 # DES (Data Encryption Standard) was standardized by NIST in 1977 and became
4 # the world's first openly standardized encryption algorithm. It is now
5 # completely broken — a 56-bit key can be exhausted in under 24 hours on
6 # consumer hardware — but it remains a vital historical and educational subject.
7 #
8 # This implementation exists to teach:
9 #
10 # 1. Feistel networks — the structural innovation that lets encryption and
11 # decryption share the same circuit (just reverse the subkey order).
12 # Only the right half passes through the round function f; the left half
13 # is simply swapped. Because f never needs inversion, decryption = same
14 # hardware, reversed keys.
15 #
16 # 2. S-boxes — the non-linear heart of DES. Each S-box maps 6 input bits to
17 # 4 output bits through a hand-crafted lookup table. Without S-boxes, DES
18 # would be entirely linear and solvable with Gaussian elimination.
19 #
20 # 3. Key schedule — how a single 56-bit key expands into 16 round keys of
21 # 48 bits each using PC-1, left rotations, and PC-2.
22 #
23 # 4. Why 56 bits is not enough — simple birthday math shows that 2^56 ≈ 72
24 # quadrillion keys can be exhausted in hours on GPU clusters.
25 #
26 # Algorithm Overview (Feistel Network)
27 # ─────────────────────────────────────
28 #
29 # plaintext (8 bytes / 64 bits)
30 # │
31 # IP (initial permutation) ← scatters bits for 1970s bus alignment
32 # │
33 # ┌── 16 Feistel rounds ───────────────────────────────────────────────┐
34 # │ L_i = R_{i-1} │
35 # │ R_i = L_{i-1} XOR f(R_{i-1}, K_i) │
36 # │ │
37 # │ f(R, K): │
38 # │ E(R) 32 → 48 bits (expansion, border bits shared) │
39 # │ XOR K_i 48-bit subkey mixed in │
40 # │ S-boxes 8 × (6 bits → 4 bits) = 32 bits out │
41 # │ P 32 → 32 bit permutation │
42 # └─────────────────────────────────────────────────────────────────────┘
43 # │
44 # FP (final permutation = IP⁻¹)
45 # │
46 # ciphertext (8 bytes)
47 #
48 # Decryption is encryption with the 16 subkeys applied in reverse order.

262 The DES key is 64 bits wide but only 56 bits are key material — bits at

273 D_i = LeftRotate(D_{i-1}, SHIFTS[i])
