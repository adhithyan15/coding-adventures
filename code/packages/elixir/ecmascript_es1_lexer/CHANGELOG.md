# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-05

### Added

- Initial implementation of ECMAScript 1 (1997) lexer
- Grammar-driven tokenization using es1.tokens
- Support for all ES1 keywords, operators, literals, and delimiters
- Convenience functions: `tokenize/1`, `load_grammar/0`, `grammar_path/0`
