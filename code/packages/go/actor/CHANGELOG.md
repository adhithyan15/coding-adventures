# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-03-22

### Added
- Message struct with immutable fields, binary wire format, factory functions
- Channel struct with append-only log, binary persistence, crash recovery
- Actor struct with mailbox, behavior function, state management
- ActorSystem with lifecycle management, message delivery, dead letters
- Comprehensive test suite with 58 test cases targeting 95%+ coverage
