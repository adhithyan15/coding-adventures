# Changelog

All notable changes to the bookmarks-extension will be documented in this file.

## [0.1.0] - 2026-03-30

### Added
- Initial release
- BookmarkStorage interface with pluggable backend architecture
- IndexedDB storage backend (default)
- Popup UI with add/edit view and bookmarks list view
- Search/filter bookmarks
- Cross-browser support (Chrome, Firefox, Safari)
- Release workflow for automated builds
