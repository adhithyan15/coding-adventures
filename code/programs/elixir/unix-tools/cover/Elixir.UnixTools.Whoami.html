5 ## What This Program Does
6
7 This is a reimplementation of the GNU `whoami` utility in Elixir. It prints
8 the user name associated with the current effective user ID.
9
10 ## How whoami Works
11
12 At its simplest:
13
14 whoami => adhithya
15
16 The program reads the `USER` environment variable to determine the
17 current user name. This is the standard approach on Unix-like systems.
18
19 ## whoami vs id
20
21 Both commands can tell you who you are, but they differ:
22
23 - `whoami` prints just the user name of the effective user ID
24 - `id` prints user ID, group ID, and all group memberships
25
26 ## whoami vs logname
27
28 - `whoami` prints the *effective* user (which changes with `su` or `sudo`)
29 - `logname` prints the *login* user (the original user who logged in)
30
31 For example:
32
33 $ whoami => adhithya
34 $ sudo whoami => root (effective user changed)
35 $ sudo logname => adhithya (login user unchanged)
36
37 ## Implementation Note
38
39 We use `System.get_env("USER")` which reads the `$USER` environment
40 variable. On most Unix systems, this is set by the login process and
41 accurately reflects the effective user. An alternative would be to use
42 `:os.cmd(~c"id -un")` to call the system `id` command, but reading
43 the environment variable is simpler and sufficient for our purposes.
