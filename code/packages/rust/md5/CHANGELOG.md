## [0.2.0] - 2026-03-23

### Added

- Full MD5 implementation from RFC 1321 in literate programming style
- `sum_md5(data: &[u8]) -> [u8; 16]` — one-shot hash function
- `hex_string(data: &[u8]) -> String` — one-shot hex digest
- `Digest` struct with streaming API: `new()`, `update()`, `sum_md5()`,
  `hex_digest()`, `clone_digest()`
- Hardcoded T-table of 64 sine-derived round constants (RFC 1321, Appendix T)
- Hardcoded S-table of 64 per-round left-rotation amounts
- Four-stage compression function (FF/GG/HH/II) with 16 rounds each
- Little-endian block parsing (`u32::from_le_bytes`) and output (`to_le_bytes`)
- MD5 padding: 0x80 + zero-fill to 56 mod 64 + 64-bit LE bit length
- Davies-Meyer feed-forward after each 64-round block compression
- 38 unit tests covering all RFC 1321 vectors, output format, little-endian
  byte order, block boundaries (0, 55, 56, 63, 64, 127, 128 bytes), edge
  cases (null byte, all-byte-values, per-byte uniqueness), and full
  streaming API coverage including clone independence and resume-after-sum
- 5 doc tests embedded in module and function documentation
