# Changelog

All notable changes to this package will be documented in this file.

## [0.2.0] - 2026-04-05

### Added
- `parse(source, version \\ nil)` — optional `version` parameter accepting
  `"es1"`, `"es3"`, `"es5"`, `"es2015"` through `"es2025"`.
  Passing `nil` (default) uses the generic grammar — backwards-compatible with v0.1.x.
- Version validation raises `ArgumentError` with a descriptive message for unknown versions.
- Expanded test suite covering all supported ES version strings, nil version, and error cases.

### Changed
- `parse/1` is now implemented as a 1-arity wrapper over `parse/2` — fully
  backwards-compatible; existing callers with one argument are unaffected.
