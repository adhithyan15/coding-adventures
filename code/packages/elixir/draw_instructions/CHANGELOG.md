# Changelog

All notable changes to this package will be documented in this file.

## [0.2.0] - 2026-03-31

### Added

- `draw_line/7` constructor for straight line segments (always stroked)
- `draw_clip/6` constructor for rectangular clipping regions with children
- Stroke support on rectangles via `stroke` and `stroke_width` options
- Font weight support on text via `font_weight` option
- Full `@type` and `@spec` annotations for all public functions
- Comprehensive literate-programming documentation with examples and tables
- Renderer `@callback` declaration for `render/1`

### Changed

- `draw_rect/6` now accepts a keyword list for the 6th argument instead of
  a plain metadata map.  This allows passing `:stroke`, `:stroke_width`, and
  `:metadata` together.  The old `draw_rect(x, y, w, h, fill, metadata_map)`
  call pattern is replaced by `draw_rect(x, y, w, h, fill, metadata: map)`.
- `draw_text/4` now accepts a keyword list for the 4th argument instead of
  a positional metadata map.  All text options (fill, font_family, font_size,
  align, font_weight, metadata) are keyword-based.
- All instruction maps now include `stroke`, `stroke_width`, and `font_weight`
  keys (set to `nil` when unused) for consistent structure.
