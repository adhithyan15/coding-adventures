5 ## What This Program Does
6
7 This is a reimplementation of the GNU `cmp` utility in Elixir. It compares

9
10 ## How cmp Works
11
12 At its simplest:
13
14 cmp file1.txt file2.txt => reports first difference (or nothing if identical)
15
16 ## Output Modes
17
18 cmp has three output modes:
19
20 | Mode | Flag | Output |
21 |-------------|------|------------------------------------------------------|
22 | Default | - | "file1 file2 differ: byte N, line L" |
23 | Verbose | -l | Lists every differing byte: "offset byte1 byte2" |
24 | Silent | -s | No output at all, just exit code |
25
26 ## Exit Codes
27
28 | Code | Meaning |
29 |------|--------------------------|
30 | 0 | Files are identical |
31 | 1 | Files differ |
32 | 2 | Error (file not found) |
33
34 ## The -b Flag
35
36 When -b (print-bytes) is combined with default output, the differing bytes
37 are shown as octal values alongside their character representations.
38
39 ## The -i Flag (ignore-initial)
40
41 Skip the first N bytes of both files before comparing. The format is a
42 single number (skip same amount in both) or SKIP1:SKIP2 (different amounts).
43
44 ## The -n Flag (max-bytes)
45
46 Compare at most N bytes, then stop. Useful for comparing only the header
47 of large files.
48
49 ## Implementation Approach
50
51 The core comparison is implemented as a pure function `compare_bytes/2`
52 that takes two binaries and options, returning a list of differences.
