# Changelog

All notable changes to this package will be documented in this file.

## [0.1.1] - 2026-03-31

### Fixed

- Fixed `GrammarParseError: Unexpected token: "puts"` by relying on the shared
  fix to `ruby.grammar`. The grammar now uses `KEYWORD` instead of `PUTS`,
  `TRUE`, `FALSE`, and `NIL`, matching what the lexer actually emits for
  keyword identifiers.
