# Changelog

## [0.1.0] - 2026-04-16

### Added

- Initial implementation of `coding-adventures-beam-bytes-decoder`.
- IFF-style `.beam` container parsing for `FOR1` / `BEAM` files.
- Version-aware decoding of atoms, code headers, imports, and exports.
- `DecodedBeamModule` model for downstream disassembly and simulation stages.
- Unit tests with synthetic `.beam` fixtures.
