# Changelog

All notable changes to this package will be documented in this file.

## [0.1.0] - 2026-04-05

### Added
- Initial SHA-512 implementation using Bitwise module for 64-bit masking
- One-shot `sha512/1` and `sha512_hex/1` functions
- Binary pattern matching for big-endian 64-bit word parsing
- Full FIPS 180-4 test vector coverage
- Block boundary and edge case tests
